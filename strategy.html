Funding Pathways Enable the Strategy

We've modeled multiple funding scenarios. Each maintains our core mission while adapting to resource availability.

🌱 Bootstrap Path (No External Funding)

How it enables the strategy: Validates core tech with minimal resources before seeking scale funding.

  • Timeline: 3-5 years to first revenue
  • Team: 1-3 people (founder + contractors)
  • Scope: Single-country pilot (Thailand), core SSOT architecture
  • Cost: ~€200K over 2 years
  • Strategy Impact: Validates Build phase; defers Scale until funding secured
  • Risk Mitigation: Focus on open-source schema to build community while bootstrapping

🏛️ Grant-Funded Path (EIC Pathfinder Success)

How it enables the strategy: Non-dilutive funding accelerates Build → Deploy → Validate without commercial pressure.

  • Timeline: 48 months (2026-2030)
  • Team: 8-12 people (consortium)
  • Scope: Multi-country pilots, full SSOT/INOS/HEMPIN stack
  • Cost: €4M (100% grant-funded)
  • Strategy Impact: Enables full Build + Deploy phases; de-risks Scale for private investment
  • Risk Mitigation: Consortium diversification; milestone-based funding
View Grant Details →

🤝 Hybrid Path (Mixed Funding)

How it enables the strategy: Balances grant de-risking with private capital agility for faster iteration.

  • Timeline: 2-3 years to product-market fit
  • Team: 4-6 people
  • Scope: Thailand pilot + lightweight EU pilot, core features only
  • Cost: €1M (€500K grant, €500K private)
  • Strategy Impact: Enables Build + partial Deploy; validates revenue model before Scale
  • Risk Mitigation: Phased feature rollout; revenue validation before scaling
