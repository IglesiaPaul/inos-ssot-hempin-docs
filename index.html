What We're Building

🌐 SSOT

Single Source of Truth — A living knowledge graph that encodes every verified actor, relationship, regulation, and value flow in the hemp economy. Machine-readable compliance for 57+ jurisdictions.

⚙️ INOS

Sovereign Operating System — Built on EU infrastructure (OVHcloud, Memgraph, Mistral AI). Zero US dependency. Event-sourced architecture for immutable, auditable truth.

🌱 HEMPIN

First Deployment — Application layer activating the ecosystem: Hemp Commons (regulatory Q&A), Hemp Swap (verified trading), Hemp Globe (geospatial map), BUD AI (natural language onboarding).

🤖 BUD AI

Natural language onboarding agent that converts farmer conversations into verified graph data. Self-hosted, sovereign AI with confidence scoring and human-in-the-loop validation.

💎 Regeneration Fund

30% of transaction value flows automatically to verified environmental cleanup projects. Embedded regenerative economics — not an add-on, but core architecture.

🌍 Solarpunk 2045

Long-term vision: Verified supply chains for all industries, sovereign technology for data autonomy, and regenerative finance at global scale. Starting with hemp.
