What We're Building

🌐 SSOT

Single Source of Truth — A multi-dimensional knowledge graph representing the hemp economy at entity, relationship, compliance, and value flow levels simultaneously.

⚙️ INOS

Operating System — Microservices + Event Sourcing + Graph Database. Sovereign infrastructure with zero US technology dependency.

🌱 HEMPIN

First Deployment — Application layer activating the ecosystem through associations, tools (Hemp Commons, Hemp Swap, Hemp Globe), and integrations.

🤖 BUD AI

Natural language onboarding agent that converts conversations into verified graph data. Self-hosted, sovereign AI infrastructure.

Regenerative token economics with 30% of value flowing to the Regeneration Fund for environmental cleanup and R&D.

🌍 Solarpunk 2045

Long-term vision: Verified supply chains for all industries, sovereign technology, and regenerative finance at global scale.
