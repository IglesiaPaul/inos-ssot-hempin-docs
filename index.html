Core Architecture

🌐 SSOT

Single Source of Truth — A living knowledge graph that encodes every verified actor, relationship, regulation, and value flow.

  • Entity layer: Canonical representation of every verified actor
  • Relationship layer: Every interaction between entities
  • Compliance layer: Per-jurisdiction regulatory encoding as machine-readable constraints
  • Value layer: Economic flow graph with embedded Regeneration Fund

⚙️ INOS

Operating System — The architecture that produces SSOT: Microservices + Event Sourcing + Graph Database + Sovereign EU infrastructure.

  • Sovereign stack: OVHcloud, Memgraph, Mistral AI (zero US dependency)
  • Event-sourced architecture for immutable, auditable truth
  • GDPR-by-design with data minimization and explicit consent flows
  • Scalable to planetary-level verified infrastructure

Tools & Mission

🤖 BUD AI

BUD AI: Natural language onboarding for verified supply chains

Natural language onboarding agent that converts farmer conversations into verified graph data. Self-hosted, sovereign AI with confidence scoring and human-in-the-loop validation.

  • Confidence scoring >0.85 threshold for auto-approval
  • Human-in-the-loop console for low-confidence extractions
  • Multi-language support (EN, FR, TH, RO + expanding)
  • Reduces onboarding friction from weeks to minutes
