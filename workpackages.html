How We Build It

Our work is organized into 6 interconnected Work Packages — each advancing the mission of verified truth infrastructure for planetary regeneration.

A Coordinated Approach

Each Work Package has clear objectives, deliverables, and timelines. Together, they form a cohesive research program designed to achieve proof-of-principle (TRL 4) by Month 48. All work is open science by default, with outputs shared via GitHub and open-access publications.

The 6 Work Packages

WP1 Project Management & Coordination

Ensures smooth execution, ethical compliance, and stakeholder alignment across the entire project.

  • Key Tasks: Consortium coordination, reporting, risk management, financial oversight
  • Deliverables: Project Handbook, periodic reports, risk register updates
  • Duration: Months 1-48 (full project)

WP2 SSOT Graph Architecture & Sovereign Infra

Designs the foundational knowledge graph and validates the sovereign EU technology stack.

  • Key Tasks: SSOT schema design (Entity/Relationship/Compliance/Value layers), sovereign stack deployment (OVHcloud, Memgraph, Mistral), performance benchmarking, security audit
  • Deliverables: SSOT Graph Schema v1.0, Sovereign Stack Benchmark Report, Security Audit Report
  • Duration: Months 1-36

WP3 BUD AI Agent Development (NL2G)

Builds the natural-language-to-graph pipeline that enables low-friction onboarding for farmers and processors.

  • Key Tasks: Ontology refinement for AI training, LLM fine-tuning (Mistral 7B), confidence scoring implementation, human-in-the-loop console development
  • Deliverables: BUD Pipeline v1.0, Confidence Scoring Model Technical Report, Human-in-Loop Console
  • Duration: Months 6-42

WP4 Compliance Engine & Regulatory Encoding

Encodes regulatory rules from 5 jurisdictions as machine-readable graph constraints for real-time compliance validation.

  • Key Tasks: Regulatory ontology mapping (TH, FR, DE, NL, AU), graph constraint encoding (SHACL/Cypher), validation against test cases, iterative expansion plan
  • Deliverables: Regulatory Ontology for 5 Jurisdictions, Compliance Engine Prototype, Validation Report
  • Duration: Months 6-48

WP5 Pilot Deployment & Validation

Validates the integrated system with 500+ verified entities in Thailand and EU test environments.

  • Key Tasks: Pilot site setup (Thailand via TiHTA LOI), user onboarding via BUD Agent, transaction validation via Compliance Engine, feedback collection and iteration
  • Deliverables: Thailand Pilot Report, EU Pilot Report, Proof-of-Principle Validation Report (TRL 4)
  • Duration: Months 12-48

WP6 Dissemination, Ethics & Open Science

Ensures open science practices, stakeholder engagement, and ethical compliance throughout the project.

  • Key Tasks: Open-access publications, GitHub repo maintenance, industry workshops, policy briefings, data management plan (FAIR principles)
  • Deliverables: 3+ Peer-Reviewed Publications, Open Source Repository, Data Management Plan, Policy Briefs
  • Duration: Months 1-48 (full project)
