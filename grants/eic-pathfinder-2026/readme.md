## ✅ Grant Application Checklist

### Pre-Application (4-8 Weeks Before Deadline)

- [ ] Identify grant opportunity and confirm eligibility
- [ ] Download application template from Funding & Tenders Portal
- [ ] Identify consortium partners (3+ countries for EIC)
- [ ] Request LOIs from partners (use `templates/loI-template.md`)
- [ ] Register all partners on Funding & Tenders Portal (get PIC numbers)
- [ ] Schedule consortium kick-off call

### Application Drafting (2-4 Weeks Before Deadline)

- [ ] Draft Section 1: Excellence (10-11 pages)
- [ ] Draft Section 2: Impact (6-7 pages)
- [ ] Draft Section 3: Implementation (5-6 pages)
- [ ] Complete budget breakdown (Excel template)
- [ ] Complete Part A online form
- [ ] Internal review (founder + advisors)

### Final Review & Submission (1 Week Before Deadline)

- [ ] External review (grant consultant if possible)
- [ ] Final edits based on feedback
- [ ] PDF formatting check (page limits)
- [ ] Submit 24+ hours BEFORE deadline
- [ ] Confirm submission receipt

### Post-Submission

- [ ] Update grant-tracker.xlsx with submission date
- [ ] Schedule follow-up reminders (evaluation timeline)
- [ ] Prepare for potential interview (EIC Transition/Accelerator)
- [ ] Continue drafting next grant application

## 🚨 Important Reminders

1. **Page Limits Are Strict:** Excess pages are automatically made invisible. Stay within limits.

3. **LOIs Are Critical:** Consortium partners must provide LOIs before submission.
