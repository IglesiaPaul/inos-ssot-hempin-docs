Executive Summary

Project: INOS/SSOT/HEMPIN — Verified Truth Infrastructure for Hemp Supply Chains
Call: EIC Pathfinder Open 2026 (HORIZON-EIC-2026-PATHFINDEROPEN-01)
Submission Deadline: May 12, 2026, 17:00 Brussels time
Requested Grant: €4,000,000 (100% funding rate, RIA topic)
Duration: 48 months (Months 1-48)
Coordinator: INOS Foundation (France)

🎯 Objective: Proof-of-principle (TRL 4) for Regulatory-Encoding Knowledge Graphs that enable proactive compliance validation in fragmented supply chains.

Core Innovation

The project converges three scientific domains to create a radically new architecture:

  • Graph Theory + Event Sourcing: A Single Source of Truth (SSOT) that encodes entities, relationships, and value flows as immutable, time-series events.
  • Regulatory Science as Code: Machine-readable encoding of 57+ jurisdictional regulations directly into the graph schema, enabling real-time compliance validation.
  • Sovereign AI Onboarding (BUD Agent): A natural-language-to-graph pipeline that allows non-technical actors to verify onto the network via conversational AI.

Expected Outcomes (Month 48)

  • ✅ Validated graph schema supporting 50+ entity types (WP2)
  • ✅ BUD Agent with >85% confidence scoring for entity onboarding (WP3)
  • ✅ Regulatory ontology for 5 jurisdictions encoded as graph constraints (WP4)
  • ✅ 500+ verified entities onboarded in Thailand + EU pilot environments (WP5)
  • ✅ 3+ open-access publications + open-source schema release (WP6)

Long-Term Vision

By 2045, this architecture extends to all industries requiring verified supply chains — organic agriculture, textiles, coffee, cocoa, fisheries. Starting with hemp, scaling to planetary regeneration infrastructure.
