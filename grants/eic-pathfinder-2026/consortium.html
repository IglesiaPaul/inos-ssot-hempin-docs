Consortium Structure

The project is coordinated by INOS Foundation (France) with prospective partners contributing domain expertise for regulatory encoding, pilot validation, and dissemination.

🇫🇷 INOS Foundation (Coordinator)

✅ Confirmed
Computer Science, AI, Supply Chain Verification

Role: Project coordination, technical leadership (WP1-WP3), sovereign stack deployment.

INOS brings core expertise in software architecture, product management, and regenerative systems design. The coordinator leads the EIC Pathfinder application and technical implementation.

Key Contributions

  • SSOT graph schema design and implementation
  • BUD AI Agent development (NL2G pipeline)
  • Sovereign EU stack benchmarking (OVHcloud, Memgraph, Mistral)
  • Project management, ethics compliance, financial reporting
