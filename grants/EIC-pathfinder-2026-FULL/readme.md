## 🎯 Project Summary

INOS/SSOT/HEMPIN builds verified truth infrastructure for the global hemp industry — a sovereign, EU-based knowledge graph that encodes:
- **Entities:** Every verified actor (farmers, processors, brands, regulators)
- **Relationships:** Every interaction (trades, contracts, certifications, samples)
- **Compliance:** 57 jurisdictions encoded as machine-readable rules
- **Value:** Economic flows (transactions, fees, regenerative fund contributions)

**Vision:** By 2045, this architecture extends to all industries requiring verified supply chains — organic agriculture, textiles, coffee, cocoa, fisheries. Starting with hemp.
