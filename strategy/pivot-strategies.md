## Executive Summary

This document outlines four pivot pathways if the primary INOS/SSOT/HEMPIN vision becomes irrelevant or unachievable. Each pivot leverages existing assets while redirecting toward viable markets.

| Pivot | Trigger | Timeline | Revenue Potential | Asset Reuse |
|-------|---------|----------|-------------------|-------------|
| **A: Crop-Agnostic** | Hemp market stagnates | 6-9 months | 3-5x hemp market | 80% |
| **B: Sovereign Infra** | Platform fails, infra demand grows | 3-6 months | €2-5M/year | 70% |
| **C: Compliance API** | Adoption slow, regulation increases | 4-6 months | €1-3M/year | 60% |
| **D: Data & Analytics** | Platform moderate, data valuable | 6-12 months | €2-10M/year | 50% |

## Pivot Decision Framework

### When to Consider Pivoting

| Signal | Severity | Action |
|--------|----------|--------|
| **Revenue <50% of target** (2 consecutive quarters) | Medium | Investigate, adjust strategy |
| **Revenue <30% of target** (3 consecutive quarters) | High | Activate pivot planning |
| **Grant success rate <15%** (vs. 25% target) | Medium | Diversify sources, consider pivot |
| **Key partnership fails** (e.g., Thailand LOI) | Medium | Find alternative, assess pivot |
| **Regulatory environment deteriorates** | High | Activate Pivot A or C |
| **Technology becomes commoditized** | Medium | Activate Pivot B |
| **Team unable to execute** | High | Restructure or pivot |
| **Funding gap >12 months** | Critical | Activate any viable pivot |

## Worst Case: Orderly Wind-Down

| Signal | Threshold |
|--------|-----------|
| All pivots fail | None viable after testing |
| Funding exhausted | <3 months runway |
| Team unable to continue | Key people depart |
| Market disappears | Regulatory ban, industry collapse |
| Legal/liability issues | Unresolvable |

### Asset Recovery Plan

| Asset | Recovery Value | Disposition Method |
|-------|---------------|-------------------|
| Hardware (Servers, GPU) | 40-60% of cost | Auction to datacenters |
| Colocation Contracts | 0% (lease) | Negotiate early termination |
| Software/IP | 20-50% of value | Sell to competitors |
| Compliance Database | 30-60% of value | Sell to regulatory tech firms |
| Graph Data | 10-30% of value | Sell to research institutions |

## Quarterly Review Checklist

```markdown
## Pivot Risk Assessment (Quarterly)

### Market Signals
- [ ] Revenue vs. target (>80% = green, 50-80% = yellow, <50% = red)
- [ ] Grant success rate (>25% = green, 15-25% = yellow, <15% = red)
- [ ] Entity growth rate (>20% QoQ = green, 10-20% = yellow, <10% = red)
- [ ] Partnership pipeline (>5 active = green, 3-5 = yellow, <3 = red)

### Team Signals
- [ ] Team morale (>4/5 = green, 3-4 = yellow, <3 = red)
- [ ] Key person retention (all retained = green, 1 departed = yellow, 2+ = red)
- [ ] Hiring success (>80% offer acceptance = green, 50-80% = yellow, <50% = red)

### Financial Signals
- [ ] Runway (>18 months = green, 12-18 = yellow, <12 = red)
- [ ] Burn rate vs. budget (<100% = green, 100-120% = yellow, >120% = red)
- [ ] Cash flow positive (yes = green, path defined = yellow, no path = red)

### Overall Assessment
- [ ] All green → Continue current strategy
- [ ] 1-3 yellow → Monitor closely, adjust tactics
- [ ] 4+ yellow OR 1+ red → Activate pivot planning
- [ ] 3+ red → Execute pivot or wind-down

### Actions
- [ ] Review completed
- [ ] Board notified
- [ ] Pivot team assigned (if needed)
- [ ] Timeline set for next review
